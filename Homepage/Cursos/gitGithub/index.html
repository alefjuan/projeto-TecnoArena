Bem-vindo ao curso de Git e GitHub!

O que é?

Git é um sistema de controle de versão distribuído usado para gerenciar alterações no código-fonte de um projeto de software. Ele permite que os desenvolvedores trabalhem em equipe em um projeto e acompanhem as alterações em diferentes versões do código-fonte. GitHub é uma plataforma de hospedagem de código-fonte na web que utiliza o Git como sistema de controle de versão. Ele permite que os desenvolvedores colaborem em projetos de software, compartilhem e revisem o código-fonte e gerenciem problemas ou problemas relacionados ao projeto.
