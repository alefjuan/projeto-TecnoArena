Bem-vindo ao curso de Node Js!

O que é?

Node.js é uma plataforma de desenvolvimento de software de código aberto baseada na linguagem JavaScript. Ele é usado principalmente para criar aplicativos de servidor escaláveis e de alto desempenho, bem como aplicativos de linha de comando. Node.js é construído sobre o mecanismo JavaScript V8 do Google Chrome, que é conhecido por ser rápido e eficiente. Ele permite que os desenvolvedores usem JavaScript no lado do servidor, tornando mais fácil para eles criar aplicativos web e APIs usando a mesma linguagem de programação em toda a pilha de tecnologia.
