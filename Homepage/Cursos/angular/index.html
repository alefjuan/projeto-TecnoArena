O que é?

Angular é um framework de desenvolvimento web em JavaScript, criado pela Google. Ele é usado para criar aplicativos web de página única (SPA) e aplicativos móveis híbridos. O Angular foi lançado originalmente em 2010 como AngularJS e foi posteriormente reescrito como Angular 2+. O Angular utiliza o padrão Model-View-Controller (MVC) para separar a lógica de negócios (model), a apresentação de dados (view) e a manipulação de eventos (controller). Ele permite que os desenvolvedores criem aplicativos web complexos usando um conjunto de ferramentas e recursos, como componentes reutilizáveis, injeção de dependência, roteamento, gerenciamento de formulários e muito mais.
